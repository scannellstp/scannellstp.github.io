NATIONAL TSING HUA UNIVERSITY, TAIWAN

Over the summer of 2017, I spent my time in Taiwan working under a Research Grant from professor Yiyu Shi at the University of Notre Dame. 8 studnets fromt Notre Dame made the trip over and we spent our time working on the topic of machine learning. I began the project by learning python using the CodeAcademy course on the language. From there, we began working on a chatbot built with tensor flow. We uesd a online server service called Codalab to increase computing power. We began looking at ways to improve the accuracy of the code, but due to the difficulty of the project we only made small changes to the source code. Codalab had restricitions on the libraries we could create while running, and therefore we attempted for several weeks to get the libraries pre-built. Once we were able to solve this issue, we had very little time to change variables in the code to improve the outputs.
